# You Can Copy And Paste v2.0

在禁止粘贴的编辑器中自由输入代码

## 快捷键

- **Shift + Esc**: 紧急停止（重载脚本）
- **Ctrl + Insert**: 强力复制
- **Shift + Insert**: 强力粘贴

## 使用方法

1. 复制需要粘贴的代码
2. 在目标编辑器中定位光标
3. 按 `Shift + Insert` 触发注入
4. 等待注入完成

## 功能特性

- 三种延迟模式：快速、标准、保守
- 自定义延迟参数
- 开机自启动选项
- 管理员权限支持
- 系统托盘控制

## 注意事项

- 注入过程中请勿操作键盘和鼠标
- 如遇问题可按 `Shift + Esc` 紧急停止
- 可在设置中调整注入速度

## 系统要求

- Windows 10/11
- AutoHotkey v2.0

## 运行

双击 `you-can-copy-and-paste-v2.ahk` 即可运行
